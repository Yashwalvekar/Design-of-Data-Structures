# Design of Data Structures

- A collection of Python-based projects developed as part of the **3rd Semester Design of Data Structures (DDS)** course. This repository demonstrates the practical implementation of fundamental data structures, algorithms, and problem-solving techniques through the following projects:

*  Tower of Hanoi Visualizer
*  Expression Calculator
*  Personal Finance Tracker

These projects showcase the application of core data structure concepts, algorithm design, and Python programming in solving real-world problems.
